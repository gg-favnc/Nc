Frequently Asked Questions

How do I add Nexus Command to my server?

You can add Nexus Command to your server by:

  1. Clicking the "Add to Discord" button on our website
  2. Selecting your server from the dropdown menu
  3. Authorizing the required permissions
  4. Completing any server-specific setup with /setup

You'll need "Manage Server" permissions to add the bot.

Why can't I use certain commands?

Command restrictions can occur for several reasons:

  • Permissions: You may lack the required role or server permissions
  • Server Settings: The command may be disabled in your server
  • Channel Restrictions: The command may be limited to specific channels
  • Cooldowns: You may need to wait before using the command again

Server admins can adjust command permissions using /settings permissions.

How do I report a bug or issue?

To report a bug or technical issue:

  1. Use the /bugreport command in Discord
  2. Submit a ticket in our support server
  3. Use the contact form on this page

Please include:

  • Detailed description of the issue
  • Steps to reproduce the problem
  • Screenshots if applicable
  • Any error messages received

What should I do if the bot is offline?

If Nexus Command appears offline:

  1. Check our status page for system updates
  2. Wait a few minutes and try again - we may be deploying updates
  3. If the outage persists, join our Discord for updates
  4. For extended outages, we'll post updates on our Twitter

Our system maintains 99.9% uptime, but occasional maintenance is required.

How can I suggest new features?

We welcome feature suggestions through:

  • Our /suggest command in Discord
  • The #suggestions channel in our support server
  • GitHub issues on our repository
  • The contact form on this page

When suggesting features, please include:

  • Detailed description of the feature
  • How it would benefit users
  • Any similar implementations you've seen
