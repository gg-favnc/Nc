Frequently Asked Questions

How do I add the bot to my server?

Click the "Add to Discord" button on our website or use the invite link from our Discord server. You'll need "Manage Server" permissions to add the bot.

Why can't I use certain commands?

Some commands require specific permissions. Check the command documentation for requirements. Server admins can adjust command permissions using /settings permissions.
